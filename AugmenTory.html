AugmenTory: A Fast and Flexible Polygon Augmentation Library

Project Information

  • Category: Augmentation Tool
  • Client: Computer Vision Developers
  • Start Date: December, 2023
  • Status: Completed

ICH Detection, Classification, Segmentation

The non-profit project "AugmenTory" supports computer vision (CV) engineers by providing the only existing library specifically designed for augmenting polygon annotations. This fast and flexible tool optimizes the transformation of polygon-based data, crucial for segmentation tasks. It offers computational efficiency and reduced storage needs, making it ideal for large-scale datasets. AugmenTory is available as an open-source tool on GitHub, facilitating easy integration into various projects
