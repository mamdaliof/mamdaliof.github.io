# Specification: Setup Conductor Environment

## Goal
Establish Conductor development framework for Farhad Hoseyni's portfolio website.

## Requirements
- Create `conductor/` configuration structure in root of the project workspace.
- Define `product.md` mapping the portfolio's vision, core page components, metrics, and key goals.
- Define `product-guidelines.md` establishing voice, tone, design principles, accessibility guidelines, and coding constraints.
- Define `tech-stack.md` detailing languages, Bootstrap framework, vendor utilities, and deployment infrastructure.
- Define `workflow.md` establishing branching models, spec/plan checkpoints, and verification/auditing steps.
- Define `tracks.md` and initial track folder structure.
