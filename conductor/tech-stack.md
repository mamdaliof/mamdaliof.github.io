# Tech Stack - Farhad Hoseyni Portfolio Website

## Programming Languages & Markup
- **HTML5**: Structured markup, using semantic elements (`<header>`, `<main>`, `<section>`, `<footer>`, `<article>`).
- **JavaScript (ES6+)**: Custom behavior scripting, navigation toggles, portfolio hover effects, and modal control.
- **CSS3 (Vanilla)**: Layout and custom visual theme definitions (`assets/css/main.css`).

## Frameworks & Layout
- **Bootstrap 5.3.x**: Responsive layout framework, grid systems, navigation styling, and modal components.

## Third-Party Libraries & Vendors
- **AOS (Animate on Scroll)**: Smooth fade-in animations on page scroll.
- **Swiper**: Touch-responsive carousel for publications or project slides.
- **GLightbox**: Lightbox interface for image zoom and project screenshots.
- **Isotope Layout**: Dynamic item filtering and masonry layout organization for the experience cards.
- **Typed.js**: Typing animation on the hero landing section.

## Development & Hosting Infrastructure
- **Local HTTP Server**: Used for testing (e.g., Python's `http.server` or Node's `http-server`).
- **GitHub Pages**: Static site hosting, linked to custom domain `mamdaliof.me` via `CNAME` registration.
- **Git Worktrees**: Branch management (e.g., development branch `setup-conductor-environment` mapped separately to verify updates without impacting master branch availability).
