Measuring Heart-rate and Oxygen Saturation in the Blood

Project Information

  • Category: Biological Signals
  • Client: Instrumentation Course project at K. N. Toosi
  • Start Date: September, 2023
  • Status: Completed

Use MAX30100 and Arduino Uno to Measure Biological Signals

As part of my Instrumentation course, I developed a project using an Arduino Uno and the MAX30100 module to measure blood oxygen levels and heart rate through infrared (IR) and red light. The project involved integrating the sensor module with the microcontroller to accurately capture and process physiological data. Through this hands-on experience, I gained insights into sensor interfacing, signal processing, and real-time data analysis. The project was well-received and earned full marks in the course.
