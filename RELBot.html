Autonomous Mobile Navigation & Visual Servoing

RELBot Tracker is a navigation and visual target tracking software stack custom-built for the RELBot mobile robot platform. It integrates inputs from wheel encoders, ultrasonic sensors, and on-board camera feeds to execute real-time localization and path correction.

The visual tracking pipeline identifies targets and guides the robot's differential drive system using a closed-loop controller. The software ensures stable tracking trajectories even during sudden obstacles, forming a robust foundation for autonomous mobile robot behaviors in dynamic indoor environments.
